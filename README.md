# 🪐 Second Mind — مجرّة Joud

نظام حياة شخصي متكامل على شكل **مجرّة تفاعلية** — بديل موسّع عن Notion بحرية أكبر.
كواكب **بكسل آرت (Pixel Art)** كبيرة على مدارات إهليلجية منظورية، وخلفية نجوم واقعية
بأسلوب صور ناسا (مولّدة بالكانفس، بدون أي ملفات خارجية)، وشمس بكسلية تتغير ألوانها حسب إنتاجيتك.

## ✨ الفكرة

- **الشمس في المنتصف** = مؤشر الإنتاجية: عالية → ذهبي، ممتازة → أبيض، جيدة → أخضر، سيئة → أحمر. تتغذى من متتبعات العادات في كل الكواكب.
- **مربع التقدّم**: أسبوعي وشهري لستة مجالات (الصحة 🟢، الصحة النفسية 🔵، التعلم الذاتي 🔷، الدراسة 🟤، الادخار والمال 🟠، المهارات 🟣).
- **مربع XP**: اكتب أهدافك (صعبة 50 / متوسطة 30 / سهلة 10 نقاط) واصعد بالرتب من ☁️ غيمة نجمية إلى 🌌 مجرّة كاملة.
- **زر (+)**: أنشئ كواكب جديدة بلا حدود، لكل جانب جديد من حياتك.

## 🌍 الكواكب

| الكوكب | الأقسام |
|---|---|
| 🟢 **الصحة** | لوحة عامة، الرياضة (سجل تمارين + جدول أسبوعي)، التغذية (وجبات + ماء + وصفات)، العناية الشخصية (روتين + تذكيرات)، الصحة النفسية (مزاج، يوميات، امتنان، تنفّس، نوم) |
| 🔵 **التعلم الذاتي والمهارات** | كورسات، القراءة (كتب + اقتباسات)، الملاحظات والمعرفة (بنك معرفة + Flashcards + روابط)، المهارات (مستويات + أهداف بخطوات) |
| 🟡 **المالية** | مصادر الدخل، المصاريف، الميزانية (حدود + تنبيهات)، الادخار (أهداف + إيداعات)، تقارير ورسوم (دونات + مقارنة شهور) |
| 🌈 **الهوايات** | هواياتي (عادة تلقائية لكل هواية)، التقدم والمهارة، مشاريعي (+ معرض أعمال)، مصادر وإلهام |
| 🟤 **الدراسة** | المواد، الواجبات، الجدول والاختبارات (+ تقويم وعد تنازلي)، ملاحظات (+ بومودورو + Flashcards)، الأهداف والدرجات، SAT، جامعات أخرى |

كل كوكب: خلفية Full-screen قابلة للتغيير من ملفاتك، قائمة جانبية مائية شفافة، ولوحة Dashboard عند الدخول.

## 🚀 التشغيل

بدون أي تثبيت أو اعتماديات — ملفات ثابتة فقط:

```bash
# افتح index.html مباشرة في المتصفح، أو:
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

أو فعّل **GitHub Pages** من إعدادات المستودع (Settings ← Pages ← Deploy from branch) ليصبح موقعك على الإنترنت.

## 💾 بياناتك

- كل شيء يُحفظ محليًا في متصفحك (localStorage) — لا حسابات ولا سيرفرات.
- من ⚙️ الإعدادات: تصدير/استيراد نسخة احتياطية JSON، تغيير الاسم والأفتار والخلفيات والعملة والأهداف.

## 🛠️ البنية

```
index.html          نقطة الدخول
css/style.css       التصميم الفضائي كاملًا
js/
  utils.js          أدوات مساعدة (تواريخ، DOM، تنسيق)
  pixel.js          محرك رسم البكسل آرت (الكواكب والشمس)
  planets.js        تعريفات الكواكب والمجالات
  store.js          الحالة + التخزين + محرك الحسابات (عادات/تقدم/XP/شمس)
  components.js     مكونات مشتركة (جداول، نماذج، عادات، رسوم، بومودورو...)
  home.js           المجموعة الشمسية والواجهة الرئيسية
  planet.js         إطار صفحة الكوكب
  views/            صفحات الكواكب الخمسة + الكواكب المخصصة
  app.js            الراوتر والتشغيل
```

JavaScript خام بدون مكتبات، RTL بالكامل، متجاوب مع الجوال.
